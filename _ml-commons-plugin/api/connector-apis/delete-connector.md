# 删除连接器

删除独立的连接器。有关更多信息，请参阅[连接器]({{site.url}}{{site.baseurl}}/ml-commons-plugin/extensibility/connectors/)。

## 路径和HTTP方法

```json
DELETE /_plugins/_ml/connectors/<connector_id>
```

#### 示例请求

```json
DELETE /_plugins/_ml/connectors/KsAo1YsB0jLkkocY6j4U
```
