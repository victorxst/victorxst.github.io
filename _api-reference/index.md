# REST API参考
**引入1.0**
{: .label .label-purple }

您可以将REST API用于OpenSearch中的大多数操作。在此引用中，我们提供了API的描述，以及包括路径和HTTP方法，支持参数以及示例请求和响应的详细信息。

该引用包括OpenSearch支持的REST API。如果缺少REST API，请提供反馈或在GitHub中提交拉动请求。
